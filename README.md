
# @increments/qiita-logo

## Install

```bash
npm install @increments/qiita-logo
```

## Usage

```js
import { qiita } from "@increments/qiita-logo"

const QiitaLogo = () =>
  <svg xmlns="http://www.w3.org/2000/svg" viewBox="{qiita.viewBox}">
    <title>Qiita</title>
    <path d="{qiita.path}"/>
  </svg>
```
